# EVSProcure Chatbot Backend - Python Dependencies
#
# ============================================================
# GEN AI PREREQUISITES (install these FIRST, before anything else)
# ============================================================
#
# 1. Install FFmpeg (required for Whisper speech-to-text):
#    - Download from: https://www.gyan.dev/ffmpeg/builds/
#    - Extract and add the bin folder to your system PATH
#    - Verify:  ffmpeg -version
#
# 2. Install Ollama (local LLM runner):
#    - Download from: https://ollama.com/download
#    - Install and run the Ollama app
#    - Verify:  ollama --version
#
# 3. Pull the Mistral model (used for intent/chat):
#    ollama pull mistral
#
# 4. Start Ollama server (if not already running):
#    ollama serve
#    Ollama API will be available at: http://localhost:11434
#
# ============================================================
# BACKEND - SETUP & RUN COMMANDS (use Python 3.12, NOT 3.13)
# ============================================================
#
# 1. Create virtual environment (first time only):
#    py -3.12 -m venv venv312
#
# 2. Activate virtual environment:
#    .\venv312\Scripts\activate
#
# 3. Install dependencies:
#    pip install --upgrade pip setuptools wheel
#    pip install -r requirements.txt
#
# 4. Run the backend server (from the backend folder):
#    .\venv312\Scripts\python.exe -m uvicorn main:app --host 0.0.0.0 --port 8000
#
#    Backend will be available at: http://localhost:8000
#
# ============================================================
# FRONTEND - SETUP & RUN COMMANDS
# ============================================================
#
# 1. Navigate to the frontend folder:
#    cd c:\EVS\EVS\updated-ui-evsprocure-UI-develop\updated-ui-evsprocure-UI-develop
#
# 2. Install dependencies (first time only):
#    npm install
#
# 3. Run the frontend dev server:
#    npm start
#
#    Frontend will be available at: http://localhost:3000
#
# ============================================================
